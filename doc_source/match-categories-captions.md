# Step 3: Match formats to categories<a name="match-categories-captions"></a>

There are different procedures to follow to create captions encodes in the output\. The correct procedure depends on the "category" that the output captions belong to\. There are five categories of captions, described in [Captions categories](categories-captions.md)\. 

On the list of outputs that you have created, make a note of the category that each captions option belongs to\.
